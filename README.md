# CampusCourses at Illinois Tech

## Project Overview

CampusCourses at Illinois Tech is a course discovery and planning project designed to help students explore academic courses, understand course details, and organize their academic planning more efficiently.

The goal of this project is to create a structured system where students can view course information, filter courses based on interests or requirements, and make better decisions while planning their semester.

## Purpose

Many students face challenges while selecting courses because course information may be scattered across different platforms or difficult to compare. This project aims to make course exploration easier by organizing important course details in one place.

## Key Features

- Store and organize Illinois Tech course information
- Search and filter courses by department, course code, or keyword
- Display course title, description, credits, and prerequisites
- Help students compare courses before registration
- Support academic planning and decision-making
