# Frontend - CampusCourses at Illinois Tech

## Overview

This folder contains the frontend code for the CampusCourses at Illinois Tech project.

The frontend is responsible for the user interface where students can search, filter, compare, and save course information.

## Main Features

- Homepage
- Course search page
- Course cards
- Filter sidebar
- Course detail page
- Saved courses page
- Responsive layout

## Folder Structure

FRONTEND/

- public/ : Static public files
- src/assets/ : Images, icons, and logos
- src/components/ : Reusable UI components
- src/pages/ : Main website pages
- src/layouts/ : Common layout components
- src/services/ : API connection files
- src/data/ : Temporary frontend data
- src/styles/ : CSS files
- src/App.jsx : Main React app component
- src/main.jsx : React entry point
